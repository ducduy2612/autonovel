# autonovel

An autonomous pipeline for writing, revising, typesetting, illustrating,
and narrating a complete novel. From a seed concept to a print-ready PDF,
ePub, audiobook, and landing page — all generated by AI agents.

The pipeline currently produces one novel (*The Second Son of the House of Bells*,
79,456 words) and is hardcoded for English output.

## Core Value

Language-configurable novel generation — a user sets a target language and
provides a seed in that language, and the entire pipeline produces output
in that language. Vietnamese (`vi`) is fully supported alongside English (`en`).

## Current State

- 27 Python scripts forming a complete seed-to-novel pipeline
- Foundation, drafting, evaluation, revision, and export phases all working
- One novel produced end-to-end (English)
- **Vietnamese language support complete (M001)**: Full pipeline produces Vietnamese output when `AUTONOVEL_LANGUAGE=vi`

  - All 15 LLM-calling scripts have language-aware prompt instructions
  - Evaluation adapted: English regex gated behind language check, language-agnostic stats preserved, LLM judge evaluates Vietnamese directly

  - 96 automated tests covering all language behaviors with zero regressions
- `run_pipeline.py` orchestrates the full pipeline with state management

## Architecture / Key Patterns

- **Python scripts** calling Anthropic API directly via `httpx`
- **Layer stack**: voice → world → characters → outline → chapters (with canon cross-cutting)
- **Two-layer evaluation**: mechanical regex slop detection (gated by language) + LLM judge scoring (language-aware cross-checks)
- **Config via `config.py`**: Single import point for all env vars (`API_KEY`, `WRITER_MODEL`, `JUDGE_MODEL`, `REVIEW_MODEL`, `API_BASE`, `FAL_KEY`, `ELEVENLABS_KEY`, `BASE_DIR`, `CHAPTERS_DIR`, `get_language()`). Reads `.env` via python-dotenv at import time. `AUTONOVEL_LANGUAGE` selects pipeline language (`en` default, `vi` supported).
- **Language helpers**: `language_instruction()` (content generators) and `analysis_language_note()` (evaluators) — return empty string for `en`, Vietnamese instructions for `vi`
- **State via `state.json`**: phase tracking, scores, debts, language field
- **Typesetting**: LaTeX (EB Garamond) + ePub via Pandoc. Vietnamese gets `\usepackage[vietnamese]{babel}` for hyphenation. EB Garamond supports Vietnamese glyphs natively.
- **Dependencies**: `httpx`, `python-dotenv` (minimal)
